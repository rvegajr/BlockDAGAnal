# Phoenix Explorer UI Design

## Design Direction
- Modern, sleek aesthetic conveying trust, transparency, cutting-edge tech
- Dark theme default with light theme option 
- Bold, high-contrast color palette

## Homepage
- Hero section with key Phoenix network stats
- Latest blocks, transactions, mini DAG viz
- Mining statistics and charts

## Block Explorer  
- Interactive, zoomable DAG visualization 
- Intuitive traversal of block tree structure
- Detailed block info on click/tap

## Transactions
- Scannable transaction details page
- Highlight addresses, amounts, status
- Recognizable icons for transaction types

## Addresses
- Summarized address history
- Tables of transactions, token balances
- Interactions with contracts/dapps

## UI Component Library
- Modern, bold, accessible components 
- Cards, tables, forms, buttons, data viz
- Consistent styles across light/dark themes
