# Phoenix Explorer Wireframes

## Homepage
- Hero section with Phoenix branding and high-level network stats 
- Navigation bar with links to key pages
- Latest blocks and transactions
- Search bar

## Block Explorer
- Visualization of block DAG
- Ability to see block details on click
- Pagination
- Filter blocks by age, miner, etc.

## Transaction Details 
- Transaction hash, status, block, timestamp
- From and to addresses + balances
- Gas details
- Raw transaction data

## Address Pages
- Address hash
- Current balance and value
- Transactions (sent and received)
- Contract info if applicable 

## Mining Statistics
- Network hashrate over time
- Average block time
- Top miners by blocks
- Mining difficulty
- Uncles count
