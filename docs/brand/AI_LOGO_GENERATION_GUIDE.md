### Iteration Process

**Step 1: Generate Initial Batch**
- Paste Prompt 1

- You'll get 4 variations

**Step 2: Upscale Favorites**
- Click `U1`, `U2`, `U3`, or `U4` (upscale buttons)
- Gets you high-resolution version (2048px+)

**Step 3: Create Variations**
- Click `V1`, `V2`, `V3`, or `V4` (variation buttons)
- Generates 4 new versions similar to selected one

**Step 4: Refine with Parameters**

Add these to your prompt for control:

```
--ar 1:1          (aspect ratio: square)
--ar 16:9         (wide format)
--v 6             (version 6, latest)
--style raw       (less artistic, more literal)
--s 50            (stylize 50, more literal)
--s 250           (stylize 250, default)
--s 750           (stylize 750, more artistic)
--q 2             (quality 2x, slower but better)
--no text         (removes any text)
--no shadows      (flat design)
--no gradients    (solid colors only)
```

**Example Refinements**:
```
/imagine [previous prompt] --no shadows --s 50 --q 2
/imagine [previous prompt] make cube more transparent, phoenix more abstract
/imagine [previous prompt] simpler lines, fewer details, app icon style
```

## üí° Pro Tips for AI Logo Generation

### 1. **Start Broad, Then Narrow**
```
Round 1: "phoenix cube logo" (explore styles)
Round 2: "geometric phoenix wireframe cube" (refine concept)
Round 3: "minimal geometric phoenix, transparent cube, orange blue" (specific)
```

### 2. **Use Style References**
```
"phoenix logo in the style of [Google/Apple/Stripe/Ethereum]"
"similar to [paste image URL]"
```

### 3. **Negative Prompting** (what to avoid)
```
--no text, realistic, shadows, gradients, complex details, cartoon
```

### 4. **Test at Small Sizes**
- Shrink to 32x32px in image editor
- If details disappear, simplify prompt
- Add "app icon style" or "favicon friendly"

### 5. **Batch Generate**
```
/imagine [prompt 1] --ar 1:1 --v 6
/imagine [prompt 2] --ar 1:1 --v 6
/imagine [prompt 3] --ar 1:1 --v 6
```
Run multiple at once, compare results

### 6. **Color Consistency**
- Always specify hex codes in prompt
- "orange (#FF6B35) blue (#4A90E2)"
- Can adjust in post-processing if needed
