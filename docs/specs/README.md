# BlockDAG Phoenix (BDP) - Technical Specifications

This directory contains authoritative specifications for the BDP stack. Each subfolder is a standalone spec with interfaces, constraints, and conformance criteria.

- core-node/: Consensus (GHOSTDAG), canonicalization, headers, block/tx formats
- evm/: EVM execution, state, transactions, gas, receipts
- rpc/: JSON-RPC (EIP-1474) conformance for BDP
- mining/: PoW algorithms (kHeavyHash, SHA-3), difficulty, rewards
- pool/: Stratum protocol, share format, pool reference architecture
- explorer/: Blockscout mapping and DAG visualization
- wallet/: Mobile and extension client specs
- sdk/: JavaScript/TypeScript, Python, Go SDKs
- devtools/: Hardhat plugin, Foundry config, Remix integration
- bridges/: LayerZero integration
- oracles/: RedStone integration
- indexing/: The Graph support
- security/: Audits, EVM/EIP conformance, fuzzing
- ops/: RPC gateway, seed nodes, monitoring
